OBS Studio Connection

sDeck needs to connect to OBS WebSocket to change scenes, toggle sources and control volume.

How to enable in OBS Studio:
  1. In OBS, go to Tools ➔ WebSocket Server Settings.
  2. Check the box "Enable WebSocket Server".
  3. Enter the port (usually 4455) and set a password.
